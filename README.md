# Share_Asso V1

## Documentation technique Symfony
Framework MVC libre écrit en PHP

### Pré-requis
- [ ] Symfony 6.0.*
- [ ] Composer
- [ ] PHP 8.0.*

### Le guide d'installation
- [ ] `composer install`
- [ ] `npm install`
- [ ] `npm run dev`
